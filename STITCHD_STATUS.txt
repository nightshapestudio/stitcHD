====================================================
ARCHITECTURE NOTES
====================================================

- STITCHD is entirely client-side: Web Audio API, Canvas, Zustand, no DB or API server.
- NO PITCH DRIFT RULE: playbackRate-based time stretch must always be labeled
  "DRAFT ONLY - changes pitch" in the UI. Non-negotiable.
- Clip arrangement lane uses pointer capture API for smooth drag.
- WAV export uses OfflineAudioContext at 44.1 kHz or 48 kHz, 16-bit PCM stereo.
- Beat grid snapping uses Shift key modifier on drag operations.
- vite.config.ts uses dedupe for react, react-dom, zustand.
